Unsupervised Clustering of Human Tissue Transcriptomes

Project Overview

This project applies unsupervised machine learning to high-dimensional gene expression data to uncover natural groupings among human tissue samples. Using transcriptomic data from the GTEx project, the goal was to cluster tissues based solely on their RNA profiles—without any prior labels, to evaluate whether underlying biological identities emerge through data alone. The outcome demonstrates how machine learning can powerfully differentiate tissue types, identify overlaps in gene regulation, and pave the way for data-driven discoveries in functional genomics.

Key Features

  • Dataset: Bulk RNA-Seq (TPM normalized) from ~19,000 GTEx samples and over 59,000 genes.
  • Gene Filtering: Removed low/no-expression genes, keeping ~20,500 for analysis.
  • PCA Reduction: Reduced data to 50 components while preserving ~80% of variance.
  • Unsupervised Algorithms:
    • K-Means: Formed clear, distinct tissue clusters.
    • GMM: Captured overlap and soft boundaries.
    • DBSCAN: Detected dense, reliable core clusters and filtered noise.
  • Evaluation Metrics: Silhouette Score, Elbow Method, Davies-Bouldin Index,
    Adjusted Rand Index (0.19) and Normalized Mutual Information (0.57) validated biological alignment.
  • Cluster Insights: High purity for tissues like testis, muscle, thyroid;
    overlap in related tissues such as arteries and adipose tissues.

Pipeline Workflow

1. Preprocessing

  • Filtered genes with low average expression (TPM < 1).
  • Ensured a clean, imputation-free dataset.

2. Dimensionality Reduction

  • Applied PCA to compress high-dimensional gene data into principal components.

3. Clustering

  • Ran K-Means, Gaussian Mixture Models, and DBSCAN.
  • Tuned hyperparameters using silhouette and elbow methods.

4. Validation & Visualization

  • Compared cluster labels with known tissue types using confusion matrices.
  • Used UMAP and t-SNE to visually confirm cluster separation and overlap.

Conclusion

This project showcases the power of unsupervised machine learning to discover biologically meaningful structures in complex genomic data. Even without labels, the clustering revealed clear patterns consistent with known tissue identities, highlighting the discriminatory power of transcriptomic profiles. The results emphasize how expression-based clustering can inform future applications in tissue classification, disease subtype discovery, and biomarker development, all through a data-first, hypothesis-free approach.
